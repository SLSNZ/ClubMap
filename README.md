# SLSNZ Club Finder

Static Leaflet club finder for Netlify/GitHub.

## Files

- `index.html` - page structure
- `styles.css` - styling and layout
- `app.js` - map/search/popup logic
- `clubs.json` - club data you can edit
- `caps.json` - club cap image links

## Editing club data

Update `clubs.json` for club names, websites, coordinates, SafeSwim links, contact details, addresses and overview text.

Update `caps.json` for club cap image URLs. The keys are normalised club names, for example `papamoa`, `mount maunganui`, or `red beach`.

## Netlify settings

- Branch: `main`
- Base directory: leave blank
- Build command: leave blank
- Publish directory: `/`
